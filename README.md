## What is MosyPaste?

MosyPaste is a Pastebin clone built with React. The tagline is *"Paste everything."* It lets you paste and share text, code snippets, or any content via a link. The site was originally hosted at `mosypaste.xyz`. This repository contains a pre-built static React app — no build step is needed to run it.

---

## Features

- **Paste anything** — text, code, logs, notes
- **Share via link** — each paste gets a unique URL
- **Clean minimal UI** — no clutter, just the paste
- **Pre-built** — no `npm install` or build step required

---

## How to Use

This is a pre-built React app. Open it directly in your browser:

```bash
start index.html      # Windows
open index.html       # macOS
```

> JavaScript must be enabled in your browser.
